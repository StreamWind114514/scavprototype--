## 注意事项
- ### 如果您在装载该补丁文件后出现以下情况：
  - 背包UI界面**鬼畜**(频繁闪动)
  - 鼠标移至物品查看详细时超长**卡顿**
  - 查看描述时频繁**闪动**
  - 或查看描述时，下方出现**大方空白内容**

- ### 可能可以解决的方法：

  - 出现这些情况 大概率是和"QOL更好的生活质量"模组产生冲突，只能**2选1**

  > 查看物品描述时频繁闪动 可以先将物品捡起来放在背包 再扔出去，就可以正常显示了，当然也可以在背包里查看

- ### 但如果将"QOL更好的生活质量"卸载后仍然存在这种情况：

那么建议**不要再使用并卸载(防止后续问题)**，这**不是您的问题**，也**不是我的问题**，而是**游戏适配性并不高**，有些人能正常用，有些人有问题

---

- ### 可解决的问题

  装载了本Json文件后，如果出现以下情况：

    - 1.进入游戏右上角不显示新语言选项，没有新增语言
    - 2.有新选项，但显示为"       v1.3"
等这种前面是空白，并且切换后没有任何中文

- ### 解决方法

  - 第1种问题：**json文件没有放置到正确的位置**，需要将它放在游戏的`Lang`文件夹内。整个游戏就只有一个"Lang"文件夹，位于`/CasualtiesUnknown_Data/Lang/`
  - 第2种问题：**缺少中文字体库**，这种需要你独自寻找中文字体MOD资源并装载，但我这里并不提供该资源文件，所以只能你自己想办法了()
