## 补丁作用

共功能性扩展翻译游戏( Demo 5.1 ) 中 大约 250 个物品描述 、90 个 功能物块的描述、170 个状态效果的描述、50 个 液体的描述等。

使用教程：下载好最新版的翻译补丁文件 (WC.json)，将该文件移动至 /CasualtiesUnknownDemoV5.1chinese/CasualtiesUnknown_Data/Lang/ 目录中，也就是游戏的语言文件夹"Lang"，接着进游戏主菜单，右上角便会多出可选择选项，标题为"维基中文v1.0"，鼠标移至上面即可显示介绍信息，点击后便可加载。

你知道吗：
翻译文件补丁文件全称 "Wiki CasualtiesUnknownDemoV5.1chinese  Chinese" ，缩减成两个字母便是"WC"，没错，我故意的wa(被打)
